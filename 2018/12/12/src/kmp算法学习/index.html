kmp算法学习

vicety · 2018-12-12 · 次阅读

速记版本

对于字符串abcabd，如果d处发生失配可以尝试从c处继续匹配
对于abc abc abd的情况如果最后一个abc失败，应该跳到第一个字母而不是上一个abd，这也就是代码中优化部分的由来。

初始状态k=-1,j=0，匹配同时++，不匹配跳转上一个next。

正文

kmp通过建立next数组，利用了匹配串中重复的部分，如abc abc abd，如果d没有匹配，那么可以尝试从c接着匹配而不用从头开始匹配。

对于未优化的版本，next数组为-100 012 345，发现尽管d未匹配可以跳到第二个abc，但如果第二个abc未匹配，仍然跳到的是必然再次失败的第一个abc，优化部分解决的就是这个问题

对于优化后的版本，next数组为-100 -100 -105

生成next数组的过程

生成next数组的过程其实是模板串自身与自身进行匹配的过程，参考正式匹配的代码可以发现较高的相似性。

public static int[] buildNext(String p) {
    int[] next = new int[p.length() + 1];
    int j = 0;
    int k = -1;
    next[0] = -1;
    while (j < p.length()) {
        if (k == -1 || p.charAt(j) == p.charAt(k)) {
            j++;
            k++;
            if (p.charAt(k) == p.charAt(j)) next[j] = next[k];
            else next[j] = k;
            // 如果没有优化，那么没有上方的if，直接next[j] = k
        } else {
            k = next[k];
        }
    }
    return next;
}

// 需要找出所有匹配的情况
public static int[] buildNextForMulti(String p) {
    int[] next = new int[p.length() + 1];
    int j = 0;
    int k = -1;
    next[0] = -1;
    while (j < p.length()) {
        if (k == -1 || p.charAt(j) == p.charAt(k)) {
            j++;
            k++;
            if(j == p.length()) next[j] = k; // 用于成功匹配到最后一个字符之后的跳转
            else if (p.charAt(k) == p.charAt(j)) next[j] = next[k];
            else next[j] = k;
        } else {
            k = next[k];
        }
    }
    return next;
}

匹配的过程

如果匹配，i++ j++；如果不匹配，跳到上一个next，在下一个while循环中重新尝试匹配；如果跳到了j == -1，说明不再有机会利用匹配串与模式串相同的前缀，此时i++,j++，对于匹配串来说是下一个字符，对于模式串来说则是回到第一个字符

// HDU 1686
public static List<Integer> kmpMatchAll(String s, String p) {
    List<Integer> ans = new ArrayList<>();
    int[] next = buildNextForMulti(p);
    int i = 0, j = 0;
    while(i < s.length()) {
        if(j == -1 || s.charAt(i) == p.charAt(j)) {
            i++;
            j++;
        } else {
            j = next[j];
        }
        if(j == p.length()) {
            ans.add(i - j);
            j = next[j];
        }
    }
    return ans;
}

public static int kmpMatch(String s, String p) {
    int[] next = buildNext(p);
    int i = 0, j = 0;
    while(i < s.length()) {
        if(j == -1 || s.charAt(i) == p.charAt(j)) {
            i++;
            j++;
        } else {
            j = next[j];
        }
        if(j == p.length()) {
            return i - j;
        }
    }
    return -1;
}
