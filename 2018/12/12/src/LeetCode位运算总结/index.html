LeetCode位运算总结

vicety · 2018-12-12 · 次阅读

文章主要对LC 371讨论区中某篇帖子中给出的位运算技巧以及应用做解释上的补充。

基本比特操作

  • 相交 A&B
  • 相与 A|B
  • 差集 A&~B
  • 取反 ~A
  • 置位 A|(1\<\<i)
  • 清除位 A|~(1\<\<i)
  • 测试位 A|(1\<\<i)!=0
  • 最低位 A&-A or A&~(A-1)
  • 最高位 （在最高位在最左侧时溢出）
long largest_power(long N) {
  //changing all right side bits to 1.
  N = N | (N >> 1);
  N = N | (N >> 2);
  N = N | (N >> 4);
  N = N | (N >> 8);
  N = N | (N >> 16);
  return (N + 1) >> 1;
}
  • 移除最低位 A&(A-1) （破坏最低位1后相与）
  • 翻转32位无符号整数（也可以改成大端序转小端序（8位））
int reverse(int n) {
n = (n & 0xaaaaaaaa) >> 1 | (n & 0x55555555) << 1;
n = (n & 0xcccccccc) >> 2 | (n & 0x33333333) << 2;
n = (n & 0xf0f0f0f0) >> 4 | (n & 0x0f0f0f0f) << 4;
n = (n & 0xff00ff00) >> 8 | (n & 0x00ff00ff) << 8;
n = (n & 0xffff0000) >> 16 | (n & 0x0000ffff) << 16;
return n;
}

考虑翻转8位数12345678，可以先将两边翻转得到43218765后左四位右移，右四位左移得到，显然是一个递归问题

简单应用

LC371 两数相加，要求不使用加减号。模拟竖式加法即可，每次提取不同的比特相加作为1，提取相同的比特进位，直到进位操作数为0为止。注意Two’s complement的设计使得无符号位运算加法可以扩展到负数，详情可以见这里。

int add(int a, int b) {
  if(b == 0) return a;
  return add(a^b, (a&b)<<1);
}

LC268 在存有0-n的长度为n的数字不相同的数组中寻找缺少的那个数字，生成0-n序列与数组全部异或即可

class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        n = len(nums)
        ans = reduce(lambda x, y : x ^ y,  range(n + 1))
        nums.append(ans)
        return reduce(lambda x, y : x ^ y, nums)

LC187 求所有出现两次及以上的长度为10的DNA片段，编码为20位，每次左移两位，与mask，或新字符，缺陷是要开一个很大的（1<<21）的数组或者用HashMap，时间复杂度上可以降低10倍

class Solution {
public:
    vector<string> findRepeatedDnaSequences(string s) {
        int sLen = s.length();
        vector<string> v;
        if(sLen < 11) return v;
        char keyMap[1<<21]{0};
        int hashKey = 0;
        for(int i = 0; i < 9; ++i) hashKey = (hashKey<<2) | (s[i]-'A'+1)%5;
        for(int i = 9; i < sLen; ++i) {
            if(keyMap[hashKey = ((hashKey<<2)|(s[i]-'A'+1)%5)&0xfffff]++ == 1)
                v.push_back(s.substr(i-9, 10));
        }
        return v;
    }
};
