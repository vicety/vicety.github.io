class Solution {
public boolean isMatch(String s, String p) {
int sLen = s.length(), pLen = p.length();
int sIdx = 0, pIdx = 0;
int starIdx = -1, sTmpIdx = -1;

while (sIdx < sLen) {
  // If the pattern caracter = string character
  // or pattern character = '?'
  if (pIdx < pLen && (p.charAt(pIdx) == '?' || p.charAt(pIdx) == s.charAt(sIdx))){
    ++sIdx;
    ++pIdx;
  }
  // If pattern character = '*'
  else if (pIdx < pLen && p.charAt(pIdx) == '*') {
    // Check the situation
    // when '*' matches no characters
    starIdx = pIdx;
    sTmpIdx = sIdx;
    ++pIdx;
  }
  // If pattern character != string character
  // or pattern is used up
  // and there was no '*' character in pattern
  else if (starIdx == -1) {
    return false;
  }
  // If pattern character != string character
  // or pattern is used up
  // and there was '*' character in pattern before
  else {
    // Backtrack: check the situation
    // when '*' matches one more character
    pIdx = starIdx+1;
    sIdx = sTmpIdx + 1;
    sTmpIdx = sIdx;
  }
}

// The remaining characters in the pattern should all be '*' characters
for(int i = pIdx; i < pLen; i++)
  if (p.charAt(i) != '*') return false;
return true;
}
}

题意：不翻转链表，完成链表求和，返回链表

解法：好像没有空间O(1)的解法，链表拿出来存到支持随机访问的容器（或者用栈）即可，另外从讨论区学到一种写法

public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
    List<Integer> lst1 = toList(l1);
    List<Integer> lst2 = toList(l2);
    ListNode ret = null;
    for (int i = lst1.size() - 1, j = lst2.size() - 1, carry = 0; i >= 0 || j >= 0 || carry != 0; i--, j--) {
        int now = 0;
        if (i >= 0) now += lst1.get(i);
        if (j >= 0) now += lst2.get(j);
        now += carry;
        carry = now / 10;
        ListNode tmp = new ListNode(now % 10);
        tmp.next = ret;
        ret = tmp;
    }
    return ret;
}

private List<Integer> toList(ListNode now) {
    List<Integer> ans = new ArrayList<>();
    while (now != null) {
        ans.add(now.val);
        now = now.next;
    }
    return ans;
}

找出数组中是否存在这样的index i j k，使arr[j] > arr[k] > arr[i]

解法一：遍历ijk中的j（也就是index from 1 to arr.length - 2），左侧维护一个最小值，右侧使用红黑树（TreeMap）每次logn查询小于arr[j]的最大值，总体nlogn

解法二：从右往左扫，维护一个以当前下标j对应的arr[j]为栈顶的单增栈，和栈中第二大元素third，每次新来的数字如果小于third，那么找到132模式，否则pop直到nums[j] > stack.top()，更新third，插入nums[j]。代码来自讨论区。

public boolean find132pattern(int[] nums) {
    int n = nums.length, top = n, third = Integer.MIN_VALUE;

    for (int i = n - 1; i >= 0; i--) {
        if (nums[i] < third) return true;
        while (top < n && nums[i] > nums[top]) third = nums[top++];
        nums[--top] = nums[i];
    }

    return false;
}

题意：问给出的字符串是否是某个子串的重复（重复至少一次）

  • 解法一：for最小重复片段长度，for每个字符是否匹配，复杂度O(n^2)
  • 解法二：利用未优化的kmp buildNext方法求next数组，如果存在循环节一定有dp[n] != 0 && dp[n] % (n-dp[n]) == 0，第二个条件保证了模板串后缀与与模板串前缀的匹配相交（避免abefgab），并且模板串长度是不相交部分的长度的整数倍（避免abcdabcdab），这样利用后缀与前缀相等的性质能够证明存在重复子串。
  • 解法三：
def repeatedSubstringPattern(self, str):
  return str in (2 * str)[1:-1]

存在循环节的情况下令循环节为A，不失一般性S=AA，上述代码构造XAAY，XY分别为A掐头去尾后的产物，显然在存在循环节的情况下成立。考虑不存在循环节的情况（这个我没想出为什么一定不存在，先咕了）
